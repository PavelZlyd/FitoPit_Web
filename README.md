# 🥗 FitoPit — Персональный план питания

> Создай сбалансированный недельный рацион за 1 клик. Учитывает цели, диету, аллергии и уровень активности.

---

## ✨ Особенности

- 🔢 **Расчёт калорий** по формуле Миффлина-Сан Жеора
- 🌍 Поддержка **русского и английского языков**
- 🌙 Тёмная и светлая темы
- 🍽 Генерация **недельного плана питания** с завтраками, обедами, ужинами
- 🚫 Учёт **аллергий и диет** (веган, без глютена и др.)
- 🔄 Возможность **перегенерации отдельных приёмов пищи**
- 📋 Кнопка **копирования плана в буфер обмена** и импорт плана из текста
- 🧠 Умный подбор: второй завтрак добавляется **только при дефиците калорий**
- 🍱 **Своя база блюд** — добавляй рецепты: КБЖУ считаются из ингредиентов
- 🧾 **Состав блюда** — кнопка «Состав» показывает ингредиенты на порцию
- 🛒 **Список покупок** — по блюдам или по продуктам (ингредиентам)
- 📚 **История планов** — сохраняй и загружай прошлые планы
- 🖨 **Печать** — распечатай весь недельный план
- 🎨 Красивый UI с плавными анимациями и фоновыми частицами
- 📱 **PWA** — работает офлайн и ставится как приложение

---

## 🚀 Как использовать

1. Открой `index.html` в браузере
   (или запусти локальный сервер: `python3 -m http.server` или `npx serve`)

2. Заполни форму:
   - Возраст, вес, рост
   - Пол, уровень активности
   - Цель: поддержание, похудение, набор
   - Диета и аллергии
   - Тип завтрака (обычный / лёгкий)

3. Нажми **🎯 Создать план**

4. Просматривай план по дням, перегенерируй отдельные приёмы или копируй весь план

### Разработка

```bash
npm install
npm run lint      # проверка кода
npm test          # unit-тесты
npm run format    # форматирование
node scripts/expand-recipes.mjs    # расширить базу блюд (без дубликатов)
node scripts/migrate-recipes.mjs   # пересчитать КБЖУ блюд из ингредиентов
```

## 📁 Структура проекта

| Файл | Назначение |
| --- | --- |
| `index.html` | Разметка, форма профиля, вкладки |
| `style.css` | Стили, темы, адаптивность, анимации |
| `data/recipes.json` | База блюд (КБЖУ, ингредиенты, выход блюда, категории) |
| `data/products.json` | Справочник продуктов: КБЖУ на 100 г сырья |
| `data.js` | Загрузка JSON, emoji-карта, обогащение картинками |
| `nutrition.js` | Расчёт КБЖУ из ингредиентов и масштабирование порций |
| `generator.js` | Логика генерации плана и балансировки калорий |
| `ui.js` | Рендеринг, i18n, импорт/экспорт, обратная связь |
| `shoppingList.js` | Агрегация списка покупок на неделю |
| `main.js` | Точка входа: тема, частицы, service worker |
| `userStore.js` | Работа с `localStorage` (профиль, блюда, план) |
| `sw.js` | Service worker для офлайн-режима |
| `scripts/migrate-recipes.mjs` | Пересчёт КБЖУ блюд из ингредиентов |
| `tests/` | Unit-тесты (Vitest) |
| `.github/workflows/` | CI и автодеплой на GitHub Pages |
| `manifest.json` | PWA-манифест |

## 💡 Технологии

- **HTML5** — структура
- **CSS3** — стили, анимации, адаптивность
- **JavaScript (ES6+)** — логика, генерация, интерактив
- **Canvas API** — анимированный фон из частиц
- **LocalStorage** — сохранение профиля, темы, языка и блюд
- **Service Worker** — офлайн-режим (PWA)
